The functionality of the chat application is that 
client 1 can send a message through the server
to client 2.

This can be sent through a different socket or address
shuold the user want to.

Additional features include a UI.
The UI REQUIRES the swing library to run
if it can't find it then it wont launch the UI.
The UI choice i made included:
 - Layout
 - Buttons for users to press
 - Text fields for users to write in
 - Background Colour
 - Font Size/Style

The messagefiled is the text a user would send over to another user.
The sendmessage function is the function that allows  the text to be sent over once the button is pressed
